# sql-business-analysis-project
SQL Business Analysis Project using MySQL and Superstore Dataset
This project focuses on analyzing retail sales data using SQL queries to identify sales trends, customer behavior, product performance, and regional business insights.
# TOOLS USED
• MySQL
• MySQL Workbench
• SQL
• Superstore Dataset
# SQL CONCEPTS USED
• SELECT
• WHERE
• GROUP BY
• ORDER BY
• COUNT
• HAVING
• INNER JOIN
• Aggregate Functions
# PROJECT FEATURES
• Category-wise Sales Analysis
• Regional Profit Analysis
• Customer Revenue Analysis
• Product Performance Analysis
• SQL Join Operations
• Business Insight Generation
# KEY INSIGHTS
• Technology category generated the highest total sales.

• Office Supplies category had the highest number of orders.

• West and East regions showed stronger profitability.

• A small group of customers contributed significantly to total revenue.

• SQL joins enabled deeper customer and product analysis.
# PROJECT OBJECTIVE
The objective of this project is to demonstrate practical SQL-based business analysis using retail sales data and analytical querying techniques.
